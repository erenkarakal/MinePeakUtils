# MinePeak Utils
An unofficial mod made to be used on MinePeak

This mod was made and tested in Fabric 1.21.10. __Requires Fabric API!__

## Features
Currently, there is only 1 feature, but more will be added depending on my mood.

### Chat Channels
Switch chat channels with a single key press, without having to type `!` or `/cc` every message.
